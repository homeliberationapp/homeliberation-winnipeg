## NO-CODE IMPLEMENTATION IN GOOGLE SITES

Since Google Sites doesn't support custom CSS, here's how to achieve the dark look:

### Method 1: Google Sites Dark Theme
```
1. In Google Sites editor, click "Themes"
2. Select a dark theme (or customize)
3. Set brand colors:
   - Primary: #0a1929 (Navy)
   - Accent: #d4af37 (Gold)
4. Adjust text colors to white/light gray
```

### Method 2: Image Backgrounds
```
1. Create dark background images
2. Set as section backgrounds
3. Ensure text is white/light colored
4. Use text boxes with transparent backgrounds
```

### Method 3: Use Embed Codes
```
For more control, embed HTML sections:
1. In Google Sites, click "Embed"
2. Select "Embed code"
3. Paste custom HTML with inline styles
4. Can include dark backgrounds, gold accents
```
